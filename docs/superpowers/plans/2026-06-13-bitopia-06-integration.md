# S6 Integration & Demo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use `- [ ]`. Read `2026-06-13-bitopia-00-overview.md` (Seams) and the spec's "Canonical demo" first. This stream runs on `main` (the human integrator), after S1/S2/S3 are merged and S4 is built. It is mostly merge + wire + verify, not TDD — but every task ends with an observable check tied to the demo's acceptance criteria.

**Goal:** Merge the parallel streams, fill the cross-stream artifacts (deployed addresses, ABIs, Blink), and make the full canonical demo run end-to-end on Sepolia with real transactions visible.

**Architecture:** Merge order S1→S2→S3 into `main`, then build S4 off that and merge. Resolve the single expected conflict (one-line registrations in `server/src/index.ts` and `client/src/App.tsx`). Bind S4's mocks to the real `worldApi`/`agentWalletOps`. Deploy contracts, publish `deployments/sepolia.json` + `shared/abi/*`, integrate Blink, then walk the 8-step demo.

**Tech Stack:** git worktrees, Hardhat (Sepolia deploy), Privy, ENS, Blink, the full app.
